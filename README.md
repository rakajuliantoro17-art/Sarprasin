# Sooko Smart Asset System

Sistem monitoring aset sekolah berbasis digital yang mendukung transparansi, partisipasi warga sekolah, dan pengambilan keputusan berbasis data.

## Features
- User Reporting (MIT App)
- Executive Dashboard
- Public Transparency Dashboard
- Real-time Data Monitoring

## Tech Stack
- Google Apps Script
- Google Sheets
- Vercel
- GitHub
- MIT App Inventor

## Architecture
User → Apps Script → Sheets → Dashboard (Executive & Public)
